# The-Awesome-Chocolates-HR-data-analysis
The report evaluates two primary areas: Workforce Composition (HR) and Sales Performance. It aims to identify where the company is growing, who its top performers are, and whether compensation is aligned with qualifications.

# Analysis Methodology


# To build this report, the following technical steps are typically taken:

Data Modeling: Connecting four primary tables: Sales, People, Products, and Geo.

DAX/Calculations: Creating measures for Total Profit, Total Boxes, and Growth Rate (Running Totals).

# Visualization Strategy:

Donut Charts: Used for quick gender/categorical split.

Box Plots: Used to show salary distribution by education.

Histogram (Bins): Grouping employees into age brackets to see population density.
